Select the 3D Model widget

Selecting a widget displays its properties in the Properties panel.

Adding 3D Data to an Experience

Add a CAD resource for this 3D Model

Click the action under the Resource property of the Model, then select the desired 3D Model file to import.

For information about supported file formats, see the Resources topic in the Help Center.

Adding 3D Data to an Experience

Position the Model

With the model selected on the canvas, position the model using the multi-colored axis. Note: If you don't see the multi-colored axis, make sure that the transform icon is selected on the canvas toolbar.

Publish the Experience

Click Publish in the ThingWorx Studio toolbar to publish the Experience to your ThingWorx Experience Service.

Click once the progress indicator next to Publish disappears.
