Using Spatial Tracking in an Experience

Getting Started

This easy step-by-step tutorial will show you how to use spatial tracking in an Experience.

Note: You can only view Studio Experiences using spatial tracking on devices that support Apple ARKit.

Watch for these hints to help you complete the steps:

Click below to get started!

Using Spatial Tracking in an Experience

Select the 3D Model widget

Click My-3D-CAD-Model in the project tree to select it.

Selecting a widget displays its properties in the Properties panel.

Using Spatial Tracking in an Experience

Select a CAD Resource for This 3D Model

Select a model from the Resource drop-down menu, or click the icon to select the desired 3D Model file to import.

For information about supported file formats, see the Resources topic in the Help Center.

Using Spatial Tracking in an Experience

Position the Model

With the model selected on the canvas, position the model using the multi-colored axis. Note: If you don't see the multi-colored axis, make sure that the transform icon is selected on the canvas toolbar.

Using Spatial Tracking in an Experience

Publish the Experience

Click Publish in the ThingWorx Studio toolbar to publish the Experience to your ThingWorx Experience Service.

Click once the progress indicator next to Publish disappears.

Using Spatial Tracking in an Experience

View the Experience

Open ThingWorx View on your mobile device.

If you don't have ThingWorx View yet, visit the Apple Store, Google Play or Microsoft Store from your mobile device to download it.

Scan the QR Code below with ThingWorx View 8.1.1 or later to launch the Experience:

LAUNCH EXPERIENCE

You forgot something! Click here to configure ThingWorx Studio. Or, you can manually enter the required project and Experience settings.
